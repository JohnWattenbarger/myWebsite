About Me

Hi I'm John Wattenbarger, a Computer Science student at the University of St. Thomas Houston. I chose to study Computer Science because I love problem solving. In 2017 I cofounded the Computer Science Club at St. Thomas with Luis Varela. I built this website to practice html, css, and javascript, as well as to showcase some of my work.

My Webpages

Resumé

This is one of the first webpages that I created. I experimented with different formatting techniques. Using CSS, I was able to recreate my resumé in the form of a webpage.

Games

I created this webpage to hold different javaScript games that I have made. The Flappy Bird game was originally created for a club event where I showed people how to make a video game from scratch.

Table Practice

I created this webpage to practice creating tables using html. I was also able to showcase one of my favorite hobbies, fantasy football. This page is a custom player rankings list that I created prior to the 2017 NFL season.

Image Mapping

With this page, I experimented with the use of images. Specifically, I practiced image mapping (creating subsections of a large image). On this page, clicking any colored section of the map will link you to the corresponding college athletics conference.

JavaScript Practice

This webpage was created when I was just starting to learn JavaScript. I experimented with simple things like changing an image, and creating a back button.
